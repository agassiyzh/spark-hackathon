# SparkGuardian 180秒 DEMO 脚本（独居老人安全场景）

## 1. 演示目标

在 3 分钟内让评委明确看到：

- 我们解决了什么真实问题（独居老人安全）
- 系统如何闭环运行（采集 -> 分析 -> 决策 -> 告警/联动）
- NVIDIA DGX Spark + 本地大模型的技术价值

## 2. 时间轴与台词（总时长 180s）

### 0s-15s：痛点与方案

**画面**：标题页 + 家庭场景背景（简洁）  
**口播**：
“独居老人家庭里，最难的是及时发现异常并做出正确响应。我们做了 SparkGuardian：以 Home Assistant 为中枢，OpenClaw 调用本地 Qwen 32B NVFP4，实现设备控制、行为分析和安全预警闭环。”

**字幕**：
- 独居老人场景：异常难发现、响应易滞后
- SparkGuardian：家庭 AI 管家（本地推理）

### 15s-35s：系统架构

**画面**：架构图（DGX Spark / Qwen / vLLM / OpenClaw / Home Assistant）  
**口播**：
“家里所有电器先接入 Home Assistant。OpenClaw 通过 API 读取设备状态并执行控制。Qwen 32B NVFP4 部署在 DGX Spark 上，负责理解指令、分析历史行为并输出建议和告警。”

**字幕**：
- Home Assistant：设备中枢
- OpenClaw：智能体执行层
- Qwen 32B NVFP4 + vLLM：本地推理核心

### 35s-70s：日常控制演示

**画面**：OpenClaw 对话窗口 + HA 状态联动  
**操作**：
1. 输入“帮我查看客厅和卧室设备状态”
2. 输入“把卧室空调调到 26 度并切到睡眠模式”

**口播**：
“先演示基础能力。AI 管家可以理解自然语言，查询当前状态并调用 Home Assistant API 执行控制，执行后会返回结果确认。”

**字幕**：
- 输入自然语言 -> 调用 HA API -> 返回执行结果

### 70s-110s：异常分析演示（核心）

**画面**：历史数据面板（活动/设备时间线）+ 对话分析结果  
**操作**：
1. 输入“分析今晚是否有异常风险”
2. 展示 AI 输出：夜间长时间无活动 + 高功耗设备持续运行

**口播**：
“这里是核心差异。系统不仅执行命令，还会结合历史行为识别异常模式，例如夜间长时间无活动且设备持续高负载，给出风险等级和解释依据。”

**字幕**：
- 行为分析：长时间无活动 + 设备异常持续运行
- 输出：风险等级 + 解释依据

### 110s-140s：告警与联动闭环

**画面**：告警卡片 + 联动动作结果 + 通知示意  
**操作**：
1. 输入“执行安全处置建议”
2. 展示动作：关闭高风险设备/发送家属提醒（按你现场可用链路演示）

**口播**：
“发现风险后，AI 管家会给出处置建议，并可一键执行联动动作，比如关闭高风险设备、推送家属通知，形成完整闭环。”

**字幕**：
- 发现风险 -> 生成建议 -> 执行联动 -> 回传结果

### 140s-165s：NVIDIA 技术点

**画面**：技术参数页（简洁列表）  
**口播**：
“我们使用 NVIDIA DGX Spark 部署 Qwen 32B NVFP4，本地 vLLM 推理，默认 32K 上下文。核心优势是低延时交互和本地隐私保护，适合家庭敏感场景。”

**字幕**：
- NVIDIA DGX Spark
- Qwen 32B NVFP4 + vLLM
- 本地推理、32K 上下文、隐私友好

### 165s-180s：总结

**画面**：总结页（价值三点）  
**口播**：
“SparkGuardian 把设备中枢、AI 推理和安全预警连接起来，为独居老人家庭提供更主动、更可解释、更可落地的智能照护方案。”

**字幕**：
- 更主动：自动发现异常
- 更可解释：给出分析依据
- 更可落地：可复现部署与演示

## 3. 镜头清单（录制检查）

- 镜头A：Home Assistant 设备总览（已接入设备）
- 镜头B：OpenClaw 对话控制（查询 + 控制）
- 镜头C：历史数据趋势（活动状态与设备状态）
- 镜头D：异常告警卡片（风险等级 + 依据）
- 镜头E：联动执行结果（设备动作/家属通知）
- 镜头F：架构与技术总结页

## 4. 现场兜底策略

- 准备一份全流程录屏备份（与口播一致）。
- 现场若模型加载或设备联动异常，30 秒内切换到备份视频继续讲解。
- 兜底视频中保留关键时间戳：70s 异常分析、110s 闭环处置、140s NVIDIA 技术点。
